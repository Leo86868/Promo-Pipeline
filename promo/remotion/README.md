# Promo Remotion Renderer

React/TypeScript Remotion project that renders the final promo video for the standalone `promo/` pipeline.

This is not a separate product surface. It is the render backend used by the Python CLIs.

## What it renders

A single composition, `HotelPromo`, producing a 1080×1920 vertical promo video from:

- clip timing data
- narration audio
- background music
- caption timestamps

Those inputs arrive as Remotion props generated by the Python side.

## How Python uses it

The Python entrypoints:

1. build props
2. stage clips and audio into `public/`
3. call `npx remotion render`
4. write the output MP4 locally

## Local commands

Install dependencies:

```bash
npm install
```

Preview in Remotion Studio:

```bash
npm run dev
```

You need a valid `props.json` in this directory first. Generate one with either:

```bash
python3 -m promo.cli.compile_promo --poi "Hotel Name" --local-clips ./my_clips
```

or

```bash
python3 -m promo.cli.smoke_local_render --local-clips ./my_clips --dry-run
```

Render directly from an existing props file:

```bash
npx remotion render HotelPromo out/manual-render.mp4 --props props.json
```
